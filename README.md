# Prochekt_Web

**Веб-платформа для постановки и выполнения задач с вознаграждением**

Проект представляет собой полноценный веб-сервис, где пользователи могут регистрироваться, создавать задачи (как заказчики), брать их в работу (как исполнители), отмечать выполнение и получать награду.

## Основные возможности

- ✅ Регистрация и авторизация пользователей
- ✅ Две роли: **Заказчик** и **Исполнитель**
- ✅ Создание задач с описанием, категорией, наградой и прикреплением файлов
- ✅ Принятие задач в работу
- ✅ Отметка задач как «Выполнена»
- ✅ Система баланса пользователей (награда списывается и начисляется)
- ✅ Генерация описания задачи с помощью **ИИ Google Gemini**
- ✅ REST API для задач
- ✅ Загрузка и скачивание файлов
- ✅ Современный и удобный интерфейс на Bootstrap 5

## Стек технологий

- **Backend**: Python + Flask
- **База данных**: SQLite + SQLAlchemy (ORM)
- **Авторизация**: Flask-Login
- **Формы**: Flask-WTF
- **Миграции**: Flask-Migrate
- **Frontend**: HTML5 + Bootstrap 5 + JavaScript
- **ИИ**: Google Gemini (gemini-1.5-flash)
- **API**: REST API
